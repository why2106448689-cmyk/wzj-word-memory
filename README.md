# wzj单词记忆

基于墨墨背单词风格，遵守艾宾浩斯背单词法的单词记忆应用（PWA）。

## 技术栈

- Next.js 15
- TypeScript
- TailwindCSS
- Zustand
- PWA

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 功能

- 首页 - 显示今日任务
- 学习模式 - 5分钟倒计时分组学习
- 快速复习 - 蓝条遮盖释义，点击揭示
- 30分钟总复习 - 随机顺序复习
- 计划表 - 艾宾浩斯排程
- 统计页 - 学习进度可视化
- List管理 - 词表总览
- PWA支持 - 可安装为手机应用
